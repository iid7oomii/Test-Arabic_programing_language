# Abjad Session Farewell Generator

A simple, interactive web application designed to generate personalized farewell messages using the syntax of **Abjad** (an Arabic programming language).

This tool was built to be used at the end of technical sessions/workshops to engage attendees, allowing them to generate a code snippet with their name and run it on the Abjad interpreter.
